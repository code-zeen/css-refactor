React Projects

여행 일지
코드 보기 사이트 바로가기

가장 첫 과제로 커스텀 컴포넌트, props, map을 사용하여 하나의 json 파일의 정보를 옮겨 담아 반복되는 컴포넌트의 정적 레이아웃을 제작했습니다.

밈 (meme) 제작기
코드 보기 사이트 바로가기

Form을 통해 받는 유저의 input을 리액트가 제어할 수 있는 제어 컴포넌트로 만들었습니다. 사용자의 텍스트를 이미지 위에 렌더링을 하는 간단한 어플입니다. 밈 이미지는 useEffect로 API를 fetch 하였고, 버튼 클릭시 무작위 이미지를 불러옵니다.

주사위 게임
코드 보기 사이트 바로가기

각각의 주사위를 오브젝트화 하여 데이터를 담았습니다. Win-condition을 충족시킬 때 (모든 주사위가 선택되었고, 모든 수가 같은 수일때) 게임을 이기고 자신의 최고기록이 남아 재도전할 수 있는 단순한 게임을 제작하였습니다.

상식 퀴즈
코드 보기 사이트 바로가기

잡상식 질문 API를 객관식 퀴즈 UI로 표현하였습니다. Check Answers 버튼으로 답을 확인 및 채점 할 수 있고, New Quiz 버튼을 눌러 새로운 질문들을 받아 올 수 있습니다.

영화 리스트 (진행중)
코드 보기 사이트 바로가기

React Router를 활용해 검색 페이지, 상세 페이지 (제작중), 그리고 마이 페이지로 조합된 싱글 페이지 어플(SPA)을 제작하였습니다. 오픈 무비(OMDb) API로 영화를 검색하여 북마크 버튼을 통해 개인페이지에 저장할 수 있습니다. 개인페이지 데이터는 sessionStorage로 브라우저창을 닫으면 데이터가 삭제되게 하였습니다. Context API를 통해 글로벌 상태/데이터 관리를 하였고 '검색 페이지' 컴포넌트가 언마운트 되어도 재빨리 마지막 검색기록으로 돌아갈 수 있습니다. 필터링 기능은 개발중입니다.
